WildFly Management Model Graphs

This site provides links to graph databases containing the WildFly management model. Each WildFly version is maintained in its own Neo4j database. Follow the links below to open the Neo4j browser of your WildFly version. The Neo4j browser contains a guide explaining the database schema and provides sample queries.
